    Connect Stripe to sell this print

    Purchase sends this print’s name, size, and price to Stripe Checkout.

    1. Get your Secret key (sk_test_... or sk_live_...) from the Stripe Dashboard → Developers → API keys. Keep it on a server only—never in this website’s JS.
    2. Deploy api/create-checkout-session.js (Cloudflare Worker) and set secret STRIPE_SECRET_KEY.
    3. Paste the Worker URL into checkoutEndpoint in assets/js/config.js.

    The browser does not need your Secret key. A Publishable key (pk_...) is optional if you redirect with the Checkout Session URL.
